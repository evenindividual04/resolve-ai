# Resolve AI Video

A Remotion video showcasing the Resolve AI platform.

## Setup

```bash
cd video
npm install
```

## Development

```bash
npm run dev
```

This opens Remotion Studio where you can preview and edit the video.

## Build

```bash
npm run build
```

This renders the video to `output/video.mp4`.

## Preview

```bash
npm run preview
```

## Video Structure

The video is 40 seconds long (900 frames at 30fps) and includes:

1. **Title Scene** (0-4s) - Project introduction
2. **Problem Scene** (4-9s) - The challenges Resolve AI solves
3. **Solution Scene** (9-12.5s) - How Resolve AI addresses these challenges
4. **Architecture Scene** (12.5-21s) - System architecture overview
5. **Features Scene** (21-25s) - Key features list
6. **Demo Scene** (25-31s) - Workflow states visualization
7. **Outro Scene** (31-40s) - Closing and call to action
